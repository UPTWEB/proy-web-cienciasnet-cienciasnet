# Módulos del Producto

| Módulo | Responsable principal | Resultado |
|---|---|---|
| Usuarios y roles | Superadmin / gestor de usuarios | Cuentas y permisos auditables |
| Estructura académica | Coordinador Académico | Periodos, matrículas, cursos y cargas |
| Asistencia alumnos | Auxiliar | Movimientos, tardanzas, faltas y anomalías |
| Asistencia y planilla docente | Yanina | Faltas, tardanzas y liquidación mensual |
| Finanzas | Yanina | Deudas, pagos completos, beneficios y reportes |
| Evaluación | Coordinador / docentes | Resultados, rankings y libretas |
| Incidencias y Psicología | Auxiliar → TOE → Psicología | Seguimiento y atención confidencial |
| Materiales y horarios | Coordinador / docentes | Recursos y calendario |
| Comunicados | Superadmin / TOE / Coordinador | Avisos segmentados y lecturas |

Los límites y reglas detalladas viven en [`../domain/`](../domain/).
El catálogo funcional completo, con reglas y casos de uso enumerados, vive en
[`../domain/use-case-catalog.md`](../domain/use-case-catalog.md).
