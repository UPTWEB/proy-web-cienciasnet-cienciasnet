# Visión del Producto

CienciasNET es una intranet web para centralizar la operación académica, administrativa, financiera y disciplinaria del
Colegio Ciencias.

## Objetivos

- Controlar entradas y salidas mediante estaciones web de reconocimiento facial.
- Gestionar periodos, matrículas, cursos, cargas académicas, evaluaciones y resultados.
- Registrar pagos manuales, beneficios y descuentos sin alterar históricos.
- Gestionar incidencias, derivaciones a TOE y atenciones confidenciales de Psicología.
- Ofrecer portales de consulta para padres, alumnos y docentes.
- Mantener auditoría y permisos estrictos para datos de menores.

## Alcance V1

1. Usuarios, roles y permisos.
2. Estructura académica.
3. Asistencia de alumnos y docentes.
4. Planilla docente.
5. Finanzas y pagos manuales.
6. Evaluaciones y resultados.
7. Incidencias, TOE y Psicología.
8. Materiales, horarios, calendario, comunicados y notificaciones.
9. Servicio facial Python, estaciones web y almacenamiento R2 privado.

## Canales

- Aplicación web responsive.
- Notificaciones dentro del panel y por correo.
- No incluye pasarela de pagos, aplicación móvil nativa, WhatsApp, SMS ni exámenes en línea en V1.
