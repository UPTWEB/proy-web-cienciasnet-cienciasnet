# ADR-002: Estaciones de Asistencia Web

**Estado:** Aceptado

Las estaciones de asistencia serán rutas web abiertas en una PC, celular o tablet. Se activan mediante QR/código temporal
desde una sesión humana segura y reciben una sesión técnica limitada.

Una PC puede usar varias cámaras; celulares/tablets normalmente una. Las estaciones nunca reciben la sesión personal del
responsable y el servicio facial Python permanece privado detrás de Laravel.
